# 訪問看護スケジュール

訪問看護ステーション向けのスケジュール管理Webアプリ。

- スタッフごとに**好きな色を無制限**に設定でき、カレンダー上で担当が一目で分かります（Googleカレンダーの色数制限なし）
- スタッフ・利用者は自由に追加・編集・削除できます
- 日／週／月表示、毎週繰り返し登録、前週コピー、担当スタッフでの絞り込み
- 日表示・週表示はGoogleカレンダー風の時間グリッド（縦軸0〜24時）。空き枠を**ドラッグ（またはタップ）するだけで予定を作成**できます
- 同時刻に重なる予定は横並びで表示され、各担当の色が一目で分かります
- 個人設定（⚙️タブ）でデフォルト表示（日／週／月）と日・週表示の時間帯を変更できます（端末ごとに保存）
- **日本の祝日に対応**：全カレンダーで日曜・祝日の日付を赤字表示し、日付の下に祝日名（薄いグレー背景）を表示。移動祝日（成人の日など）・春分・秋分・振替休日・国民の休日も自動計算します
- 日付の下に**終日イベント枠**（時間に関係ない予定：会議・休業日など）を追加できます。祝日名の下にグレーのチップで表示されます
- スマホ対応（レスポンシブ）
- データは各端末のブラウザ内（localStorage）に保存。JSONエクスポート／インポートでバックアップ・共有

## 使い方

https://nknrmk8.github.io/visit-nurse-scheduler/ をブラウザで開くだけ。サーバーやログインは不要です。

初回は「スタッフ」「利用者」タブで登録してから、「スケジュール」タブで予定を追加してください。「データ」タブからサンプルデータの投入もできます。
